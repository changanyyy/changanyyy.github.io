1 正则表达式，正则运算

下面是算术和计算理论的区别

arithmetic theory of computation
objects numbers languages
tools \(+\), \(\times\) \(U,*,^\circ\)

正则运算被用来：

  1. 设计识别特定语言的自动机
  2. 证明特定的语言不是正则的

1-1 正则运算

正则运算有三种，合并，连接，Star，详细见下面定义。

定义1. Let A and B be languages. We define the regular operations union, concatenation, and star as follows:

  • Union: \(A∪B=\{x|x∈A\,or\,x∈B\}\)
  • Concatenation: \(A\circ B=\{xy|x\in A\,and\,y\in B\}\)
  • Star: \(A^*=\{x_1x_2...x_k|k\ge 0\,and\,each\,x_i\in A\}\)

下面对三种运算，我们有三个定理说明正则运算是封闭的。

1-1-1 Union

定理1. The class of RL is closed under union operation.

下面证明一下：

Let \(N_1=(Q_1,\Sigma,\delta_1,q1,F1)\) recognize \(A_1\), \(N_2=(Q_2,\Sigma,\delta_2,q2,F2)\) recognize \(A_2\), \(N_1\), \(N_2\) is NFA.

Construct \(N=(Q,\Sigma, \delta, q0, F)\) to recognize \(A_1\cup A_2\).

  1. \(Q=\{q_0\}\cup Q_1 \cup Q_2\)

  2. The state \(q_0\) is the start state of N

  3. The accept state \(F=F_1\cup F_2\)

  4. \[ \delta (q,a)=\left \{ \begin{array}{c}\delta_1(q,a),q\in Q_1 \\ \delta_2(q,a),q\in Q_2 \\ \{q_1,q_2\},q=q_0\,and\,a=\epsilon\\ \emptyset,q=q_0\,and\,a\ne\epsilon \end{array}\right. \]

    其中\(q\in Q\)，\(a\in\Sigma_\epsilon\)

状态转移图如下图，实际上就是构造一个NFA能同时识别\(A_1\)和\(A_2\)。

1-1-2 Concatenation

定理2. The class of RL is closed under concatenation operation.

证明如下：

Let \(N_1=(Q_1,\Sigma,\delta_1,q_1,F_1)\) recognize \(A_1\), \(N_2=(Q_2,\Sigma,\delta_2,q_2,F_2)\) recognize \(A_2\), \(N_1\), \(N_2\) is NFA.

Construct \(N=(Q,\Sigma, \delta, q_1, F)\) to recognize \(A_1\circ A_2\).

  1. \(Q=Q_1 \cup Q_2\)

  2. The state \(q_1\) is the start state of N

  3. The accept state \(F=F_1\cup F_2\)

  4. \[ \delta (q,a)=\left \{ \begin{array}{c} \delta_1(q,a),q\in Q_1\,and\,q\notin F_1\\ \delta_1(q,a),q\in F_1\,and\,q\ne\epsilon \\ \delta_1(q,a)\cup\{q_2\},q\in F_1\,and\,q=\epsilon\\ \delta_2(q,a), q\in Q_2 \end{array} \right. \]

    其中\(q\in Q\)，\(a\in\Sigma_\epsilon\)

1-1-3 Star

定理3. The class of RL is closed under star operation.

证明如下：

Let \(N_1=(Q_1,\Sigma,\delta_1,q_1,F_1)\) recognize \(A_1\).

Construct \(N=(Q,\Sigma, \delta, q_0, F)\) to recognize \(A_1^*\).

  1. \(Q=Q_1 \cup \{q_0\}\)

  2. The state \(q_0\) is the new start state of N

  3. The accept state \(F=F_1\cup \{q_0\}\)

  4. \[ \delta (q,a)=\left \{ \begin{array}{c} \delta_1(q,a),q\in Q_1\,and\,q\notin F_1\\ \delta_1(q,a),q\in F_1\,and\,q\ne\epsilon \\ \delta_1(q,a)\cup\{q_1\},q\in F_1\,and\,q=\epsilon\\ q_1, q=q_0\,and\,a=\epsilon \end{array} \right. \]

    其中\(q\in Q\)，\(a\in\Sigma_\epsilon\)

实际上就是在接受状态通过空转移返回到开始状态。并且由于可以接收空串，所以起始状态也应该是接收状态，即接受\(\epsilon\)，但是单纯地把起始状态标记为接收状态会发生一些问题（如果不是从接受状态转移到起始状态的话，那么这时不可接受！），所以应该把接受\(\epsilon\)的起始状态单独提取出来。

1-2 正则表达式

下面是正则表达式定义：

定义2. Given an alphabet \(\Sigma\), R is a regular expression if

  1. \(R=a\), with \(a\in \Sigma\) ; denoting the languages {a} .
  2. \(R=\epsilon\) ; denoting the languages {ε } .
  3. \(R=\emptyset\) ; denoting the languages \(\emptyset\).
  4. \(R=(R_1+R_2)\), with \(R_1\) and \(R_2\) regular expressions;denoting the languages \(L(R_1)\cup L(R_2)\).
  5. \(R = (R_1\circ R_2)\), with \(R_1\) and \(R_2\) regular expressions; denoting the languages \(L(R_1)L(R_2)\).
  6. \(R = (R_1^*)\), with \(R_1\) a regular expression; denoting the languages \(L(R1)^*\).

1-2-1 FA和RE等价

定理4. A language is regular if and only if some regular expression describes it.

想要证明这个定理，我们需要从两个方向入手，

1-2-1-1 证明1

先证明正则表达式表示的语言是正则语言，证明的要点就是把正则表达式表达的语言用相应的自动机来表示：

  1. 对于\(R=a\)，\(L(R)= \{ a \}\)，可以用如下自动机来识别它

    形式化构造就是 \(N=( \{ q_1, q_2 \}, \Sigma, \delta, q_1, \{ q_2 \} )\)

    • 状态转移方程是\(\delta(q_1,a)=\{q_2\}\)，其余都是\(\emptyset\)

  2. 对于\(R=\epsilon\)，\(L(R)=\{\epsilon\}\)，下面的自动机可以识别它

    形式化构造就是 \(N=( \{q_1\},\Sigma,\delta,q_1,\{q_1\})\)

    • 状态转移方程是对于所有r和b，\(\delta(r,b)=\emptyset\)

  3. 对于\(R=\emptyset\)，\(L(R)=\emptyset\)，下面的自动机识别它

    形式化构造就是 \(N=(\{q_1\},\Sigma,\delta,q_1,\emptyset)\)

    • 状态转移方程是对于所有r和b，\(\delta(r,b)=\emptyset\)

  4. 对于\(R=(R_1+R_2)\)，\(R = (R_1\circ R_2)\)和\(R = (R_1^*)\)，证明已经在定理1,2,3中给出。

  5. （还有一种证明方法是归纳证明，进行递归地证明即可，跟这个类似。）

1-2-1-2 证明2

下面证明正则语言可以被正则表达式描述

证明方法1（GNFA法）：

Let M have k states \(Q=\{q_1,...,q_k\}\)

  1. Add two states（加首尾）

    • Connect \(q_{start}\) to earlier \(q_i\)
    • Connect old accepting states to \(q_{accept}\)

  2. Complete missing transitions by \(\emptyset\)（补空边)

  3. Join multiple transitions（并标号）

  4. Rip the internal states, one by one

    1. Removing state \[ q_{rip} \in Q-\{q_{start}\}-\{q_{accept}\}\,Q'=Q-\{q_{rip}\} \] （逐渐减少内态）

    2. Changing the transition function \(\delta\) by \[ \delta'(q_i,q_j)=\delta(q_i,q_j)+\delta(q_i,q_{rip})(\delta(q_{rip},q_{rip}))^*\delta(q_{rip},q_j) \] for every \(q_i\in Q'-\{q_{accept}\}\) and \(q_j\in Q'-\{q_{start}\}\)

  5. if k>2 goto 4, else return R

证明方法2：

设DFA \(M(\{q_1,q_2,...,q_n\},\Sigma,\delta,q_1,F)\)接受语言L，对M的状态进行编号。在此基础上，引入记号\(R_{ij}^k\)，它是字符串的集合，定义如下： \[ R_{ij}^k=\{x|\delta(q_i,x)=q_j,且中间不经过编号大于k的状态，但i,j可以大于k,x\in \Sigma^*\} \] 注意：x是字符串。k的意义是：对于x的一切非空真前缀y，\(\delta(q_i,y)=q_m\)，其中\(m\le k\)。

基本思想：引入\(R_{ij}\)的递归定义，即 \[ R_{ij}^k=R_{ij}^{k-1}+R_{ik}^{k-1}(R_{kk}^{k-1})^*R_{kj}^{k-1} \] ，感觉有点像动态规划哈哈哈哈。

\(R_{ij}^k\)的递归定义如下：

  1. \[ R_{ij}^0=\{a|a\in \Sigma, 且\delta(q_i,a)=q_j\}, (i\ne j) \]

  2. \[ R_{ij}^0=\{a|a\in \Sigma, 且\delta(q_i,a)=q_j\}\cup\{\epsilon\}, (i=j) \]

  3. \[ R_{ij}^k=R_{ij}^{k-1}+R_{ik}^{k-1}(R_{kk}^{k-1})^*R_{kj}^{k-1},(k=1,2,...,n) \]

下面可以开始真正的证明了 \[ L(M)=\underset{q_f\in F}{\cup}R_{1f}^n \] 其中\(L(M)\)就是DFA M识别的语言，下面的问题就是如何把$L(M)$用正则表达式表示呢？

现在证明，对于任何 \(R_{ij}^k\) ，存在正则表达式 \(r_{ij}^k\) 可以代表 $R_{ij}^k $ 。

归纳基础：k=0。 \(R_{ij}^0\) 是一个有穷集合，其中每个元素是 \(\Sigma\) 中的符号或空串，因此 \(r_{ij}^0\) 可以写成 \(a_1+a_2+...+a_p（i \ne j）\) 或者 \(a_1+a_2+...+a_p+\epsilon(i=j)\) 的形式。这里， \(\{a_1,a_2,...,a_p\}\) 是让 \(\delta(q_i,a)=q_j\) 的一切符号a的集合。

归纳步骤：设对\(m<k\)的一切m，都已经求出正则表达式\(r_{ij}^m\)代表\(R_{ij}^m\)，现在考虑m=k。根据\(R_{ij}^k\)递归定义，存在正则表达式\(R_{ij}^k=R_{ij}^{k-1}+R_{ik}^{k-1}(R_{kk}^{k-1})^*R_{kj}^{k-1},(k=1,2,...,n)\)代表\(R_{ij}^k\)。

证毕。

1-2-2 FA和RG的等价性

1-2-2-1 FA接受的语言是正则语言

证明：根据FA M构造RG G。基本思想是让RG的派生与DFA的状态转移相对应。

  1. 设\(DFA\,M=(Q,\Sigma,\delta,q_0,F)\)，取右线性\(G=(Q,\Sigma,P,q_0)\)，\(p=\{q\rightarrow qp|\delta(q,a)=p\}\cup \{q\rightarrow a|\delta(q,a)=p,p\in F\}\)

  2. 证明\(L(G)=L(M)-\{\epsilon\}\)

    对于\(a_1a_2...a_{n-1}a_n\in \Sigma^+\)，\(a_1a_2...a_{n-1}a_n\in L(G)\)

    \(q_0\Rightarrow^+a_1a_2...a_n\)

    \(\Leftrightarrow q_0\rightarrow a_1q_1,\,q_1\rightarrow a_2q_2,\,...\)

    \(\Leftrightarrow \delta(q_0,a_1)=q_1,\delta(q_1,a_2)=q_2,...\)

    \(\Leftrightarrow \delta(q_0,a_1a_2...a_n)=q_n\in F\)

    \(\Leftrightarrow a_1a_2...a_n\in L(M)\)

  3. 关于空字符串

    • 如果\(q_0\notin F\)，那么\(\epsilon \notin L(M)\)，\(L(G)=L(M)\)。
    • 如果\(q_0\in F\)，那么扩充G为G'，\(L(G')=L(G)\cup\{\epsilon\}=L(M)\)。其中，G'比G增加一个开始符号S和两个产生式\(S\rightarrow q_0|\epsilon\)。

  4. 综上所述，对于任意DFA M，存在正则文法G，使得\(L(G)=L(M)\)，定理得证。

1-2-2-2 正则语言可以由FA接受

证明：根据RG，构造FA。基本思想是让FA模拟RG的派生过程。

  1. 设\(G=(V,T,P,S)\)，且\(\epsilon\notin L(G)\)，取\(FA\,M(V\cup\{f\},T,\delta,S,\{f\}),f\notin V\)。

    定义产生式如下

    \(\delta(A,a) = \begin{cases} \{B|A\rightarrow aB\in P\}\cup\{f\}, & if\, A\rightarrow a\in P \\ \{B|A\rightarrow aB\in P\}, & if\,A\rightarrow a\notin P \end{cases}\)

  2. 证明\(L(M)=L(G)\)

    对于\(a_1a_2...a_n\in T^+\)，

    \(a_1a_2...a_n\in L(G)\Leftrightarrow S\Rightarrow^+a_1a_2...a_n\)

    \(\Leftrightarrow S\Rightarrow a_1A_1\Rightarrow a_1a_2A_2\Rightarrow...\)

    \(\Leftrightarrow S\rightarrow a_1A_1,A1\rightarrow a_2A_2,...,A_{n-1}\rightarrow a_n\in P\)

2 正则表达式的代数定律

2-1 交换律和结合律

设L，M和N是正则表达式，则：

  1. \(L+M=M+L\)，并的交换律
  2. \((L+M)+N=L+(M+N)\)，并的结合律
  3. \((LM)N=L(MN)\)，连接的结合律
  4. 连接自然没有交换律...

2-2 单位元和零元

单位元：当运算符作用于单位元和某个其它值时，结果等于其他值。

零 元：当运算符作用到零元和其它值时，结果是零元。

  1. \(\emptyset+L=L+\emptyset=L\)，所以\(\emptyset\)是并运算的单位元
  2. \(\epsilon L=L\epsilon=L\)，所以\(\epsilon\)是连接运算的单位元
  3. 并运算没有零元
  4. \(\emptyset L=L\emptyset=\emptyset\)，所以\(\emptyset\)是连接运算的零元

2-3 分配率

  1. \(L(M+N)=LM+LN\)，连接对于并的左分配律
  2. \((M+N)L=ML+NL\)，连接对于并的右分配律

2-4 幂等律

幂等：一个运算符作用到两个相同的参数值， 结果还是那个值

  1. \(L+L=L\)，并的幂等律

2-5 和闭包有关的定律

  1. \((L^*)^*=L^*\)
  2. \(L(\emptyset^*)=L(\epsilon)=\{\epsilon\}\)，空表达式的闭包和\(\epsilon\)等价，注意不要弄混正则表达式和语言，正则表达式是一个串，而语言是一个集合
  3. \(\epsilon^*=\epsilon\)，多个空串连接还是空串，这里注意和上面2的区别
  4. \(L^+=LL^*=L^*L\)，\(L^+\)表示1个或多个\(L\)
  5. \(L^*=L^++\epsilon\)

4 正则表达式的泵引理

泵引理运用了鸽巢原理，在知识补充模块。

4-1 Property of regular languages

L is a regular language \(\Rightarrow \exists DFA\,A:L(A)=L\)

Let \(A=(Q,\Sigma,\delta,q_0,F)\), and \(n=|Q|\)

Get \(w\in L\), and suppose \(w=a_1a_2...a_m,m\ge n\)

Let \(q_i=\overset{-}{\delta}(q_0,a_1a_2...a_i)\)

​ \(\Rightarrow\exists0<i<j\le n\)

下面是泵引理的形象标识，用状态转移图。

Pumping lemma: For each regular language L, there is a pumping length p, such that for any string \(s\in L\) and \(|s|\ge p\), we can write s=xyz with

  1. \(xy^iz\in L\) for every \(i\in \{0,1,2,...\}\)
  2. \(|y|>0\)
  3. \(|xy|\le p\)

注意，上面有一些暗含的意思：

  1. 暗示\(xz\in L\)
  2. y不能是空串
  3. 不是永远成立

经得起泵测试是RL的必要条件（不是充分条件）

证明一下：

设\(p=|Q|=n\)，\(s=a_1a_2...a_m\)，其中\(m>n\)；\(q_f\)是可接受状态。

根据鸽笼原理，上面一定有两个状态相同：\(q_i = q_j\)，其中\(0<i<j\le n\)

4-2 一个小例子

证明一个语言不是正则的，可以证明它不满足泵引理。

题目：Prove \(B = \{0^n1^n | n\ge 0\}\) is not regular.

答：

  1. Assume that B is regular （反证法）

  2. Let p be the pumping length, and \(s=0^p1^p\in B\)，

    \(s = xyz = 0^p1^p\), with \(xy^iz\in B\) for all \(i\ge 0\). Three options for y:

    1. \(y=0^k\), hence \(xyyz=0^{p+k}1^p\notin B\)
    2. \(y=1^k\), hence \(xyyz=0^p1^{k+p}\)
    3. \(y=0^k1^l\), hence \(xyyz=0^p1^l0^k\)

  3. Conclusion: The pumping result does not hold, the language B is not regular.

5 正则语言的封闭性

定义. 如果属于某个语言类的任何语言在某个特定运算下 所得的结果仍然属于该语言类,则称该语言类对这个运算是封 闭的，并称该语言类对这个运算具有封闭性。

  • 两个正则语言的并是正则语言。
  • 两个正则语言的连接是正则语言。
  • 正则语言的闭包是正则语言。
  • 两个正则语言的交是正则语言。
  • 正则语言（对全集）的补是正则语言。
  • 两个正则语言的差是正则语言。
  • 正则语言的逆转是正则语言。
  • 正则语言的同态是正则语言。
  • 正则语言的逆同态是正则语言。

6 正则语言的可判定性

定理1. 设\(DFA\,M=(Q,\Sigma,\delta,q_0,F)\)，\(L=L(M)\)非空的充要条件是：存在\(x\in \Sigma^*\)，\(|x|<|Q|\)，\(\delta(q_0,x)\in F\)。

定理2. 设\(DFA\,M=(Q,\Sigma,\delta,q_0,F)\)，\(L=L(M)\)为无穷的充要条件是：存在\(x\in \Sigma^*\)，\(|Q|\le|x|<2|Q|\)，\(\delta(q_0,x)\in F\)。

定理3. 设\(DFA\,M_1=(Q_1,\Sigma,\delta_1,q_{01},F_1)\)，\(DFA\,M_2=(Q_2,\Sigma,\delta_2,q_{02},F_2)\)，那么存在判定\(M_1\)与\(M_2\)是否等价的算法

定理4. 设L是字母表上的RL，对于任意的\(x\in \Sigma^*\)，存在判定x是不是L的句子的算法。

7 自动机等价性和最小化

对于一些状态比较多的自动机，我们可能会出现冗余状态，这时就会出现化简的问题。但是一旦自动机比较复杂的话，就很难用肉眼来进行化简，所以我们需要用形式化的方法来确定一个化简算法。

定义. 能引导FA从开始状态\(q_0\)到达状态q的字符串集合为：

​ \(set(q)=\{x|x\in \Sigma^*,\delta(q_0,x)=q\}\)

定义. DFA M确定的\(\Sigma^*\)上的等价关系\(R_M\)，\(M=(Q,\Sigma,\delta,q_0,F)\)，对于\(\forall x,y\in \Sigma^*\)，\(xR_My\Leftrightarrow\delta(q_0,x)=\delta(q_0,y)\)，显然，\(xR_My\Leftrightarrow \exists q\in Q,xy\in set(q)\)（根据上面定义得到）。

两个字符串等价的直观含义就是：从初始状态\(q_0\)出发，x和y都能把自动机M引导到相同的状态q，\(q\in Q\)。

定义. 语言L确定的\(\Sigma^*\)上的关系\(R_L\)，对于\(\forall x,y\in\Sigma^*\)，\(xR_Ly\Leftrightarrow\)(对\(\forall z\in\Sigma^*,xz\in L\Leftrightarrow yz\in L\))。也就是说对于的\(xR_Ly\)，在x和y后面无论接字符串表中的任何串z，xz和yz要么都是L的句子，要么都不是L的句子。（L不一定是正则的）

我们可以易得，满足\(R_M\)就一定满足\(R_L\)。

定义. 右不变（right invariant）等价关系。设R是\(\Sigma^*\)上的等价关系，对于\(\forall x,y\in\Sigma^*\)，如果\(xRy\)，那么必有\(xzRyz\)，对于\(\forall z\in\Sigma^*\)成立，那么称R是右不变的等价关系。（这里的R不一定是\(R_M\)也不一定是\(R_L\)）。

定义. 关系R的指数——设R是\(\Sigma^*\)上的等价关系，则称\(|\Sigma^*/R|\)是R关于\(\Sigma^*\)的指数，简称为R的指数。注意：\(|\Sigma^*/R|\)是指关系对\(\Sigma^*\)的划分。R的一个等价类——\(\Sigma^*\)的关于R的一个等价类，也就是\(\Sigma^*/R\)的任意一个元素（集合），简称为R的一个等价类。

定理. （Myhill-Nerode定理）下列三个命题等价：

  • \(L\subseteq\Sigma^*\)是RL
  • L是\(\Sigma^*\)上的某一个具有有穷指数的右不变等价关系R的某些等价类的并
  • \(R_L\)具有有穷指数

7-1 极小化算法

  1. 为所有状态对\((p,q)(p,q\in Q)\)画一张表，开始时表中每个格子均为空白（未做任何标记）

  2. 对\(p\in F,q\notin F\)的一切状态对(p,q)，在相应的格子中作出标记（画个x），表示(p,q)是可以区分的

  3. 重复下述过程，直到表中内容不在改变为止：

    如果存在一个未被标记的状态对(p,q)，且对于某个\(a\in\Sigma\)，如果（\(r=\delta(p,a),s=\delta(q,a)\)）已做了标记，则在(p,q)相应的格子内做标记

  4. 在完成123之后，所有未被标记的状态对(p,q)都是等价的，即\(p\equiv q\)，状态p和状态q可以合并

8 知识补充

8-1 GNFA

广义非确定性有穷自动机定义如下：

定义3. A Generalized nondeterministic finite automaton(GNFA) is a 5-tuple \((Q,\Sigma,\delta,q_{start},F_{accept})\), where

  1. Q is the finite state set
  2. \(\Sigma\) is the input alphabet.
  3. \(q_{start}\) is the start state.
  4. \(q_{accept}\) is the accept state.
  5. \(\delta:(Q-\{q_{accept}\})\times (Q-\{q_{start}\})\rightarrow R\)

\(R\) is the set of regular expressions over \(\Sigma\)

\(\delta(q_i,q_j)=R\Leftrightarrow\) from \(q_i\) to \(q_j\) has the RE R as the label, R\(\in R\)

GNFA的特点：

  1. The interior \(Q-\{q_{accept} , q_{start}\}\)

    is fully connected by \(\delta\) .（全连接）

  2. From \(q_{start}\) only "out going transitions"（只出不回）

  3. To \(q_{accept}\) only "ingoing transitions"（只回不出）

  4. Impossible \(q_i\rightarrow q_j\) transitions are "\(\delta(q_i,q_j)=\emptyset\)"

  5. \(q_{accept} \ne q_{start}\)

GNFA和一般NFA的唯一区别：GNFA的边推广为RE（子自动机），而不是单个字母或者空转移。

对GNFA化简，最后化简成只剩下起始状态和接收状态，这个对应的边就是对应的正则表达式。

8-2 一些术语

  • RL: Regular Language
  • RG: Regular Grammar
  • RE: Regular Expression
  • FA: Finite Automaton

8-3 鸽巢原理Pigeonhole principle

If m pigeons are placed into fewer than m holes, some hole has to have more than one pigeon in it.

8-4 关系

  • 二元关系（是一个集合）

    • 任意的\(R\in A\times B\)，R是A到B的二元关系
    • \((a,b)\in R\)，也可以表示为：\(aRb\)
    • A称为定义域（domain），B称为值域（range）
    • 当\(A=B\)时，称R是A上的二元关系

  • 二元关系的性质

    自反性（reflexive），反自反性（irreflexive），对称性（symmetric），反对称性（asymmetric），传递性（transitive）

  • 等价关系（equivalence relation）

    • 具有自反性，对称性，传递性的二元关系
    • 比如“=”

  • 等价类（equivalence class）（由等价关系R决定）

    S要满足如下要求的划分：\(S_1,S_2,S_3,...,S_n,...\)称为S关于R的等价划分，\(S_1\)称为等价类。

    1. \(S=S_1\cup S_2\cup...\cup S_n\cup ...\)
    2. 如果\(i\ne j\)，那么\(S_i\cap S_j=\emptyset\)
    3. 对任何的i，\(S_i\)中的任意两个元素a，b，\(aRb\)恒成立
    4. 对任意的i，j，\(i\ne j\)，\(S_i\)中的任意元素a和\(S_j\)中的任意元素b，\(aRb\)横不成立

    等价关系R将S分成的等价类个数称为R在S上的指数

8-5 \(R_L,R_{L(M)},R_M\)的关系

  1. 如果\(xR_My\)，那么一定\(xR_Ly\)，反之不一定。
  2. \(R_{L(M)}\)中的M是DFA，\(L(M)\)是正则语言，但是\(R_L\)中的L不一定是正则的。
